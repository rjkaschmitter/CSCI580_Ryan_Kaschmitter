In order to run my code you need to have Jupyter Notebook installed and the ipynb file in the current directory

After you have it installed you run the command 'jupyter notebook'

What you do is follow one of the two links under the "Or copy and paste one of these URLS:" section to the jupyter notebook webpage. You then run the code by selecting the first cell of code and clicking the little play button at the top of screen multiple times until you get to the very bottom.
